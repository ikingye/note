      • 环境因素(Environmental)

    • 波特五力模型
      • 供应商的议价能力
      • 新进入者的威胁
      • 同业竞争者的竞争程度

    • 波特三种竞争战略
      • 总成本领先战略（Overall cost leadership）
      • 差别化战略（Differentiation）
      • 专一化战略（Focus）
    • 波士顿矩阵
      • 明星（高增长、高市场占有率）
      • 金牛（低增长、高市场占有率）
      • 问题（高增长、低市场占有率）
      • 瘦狗（低增长、低市场占有率）
    • 麦肯锡矩阵
      • 竞争力高、行业吸引力高：力保优势
      • 竞争力高、行业吸引力中：争取领先
      • 竞争力中、行业吸引力高：争取好转
      • 竞争力低、行业吸引力高：加倍投资或撤退
      • 竞争力中、行业吸引力中：维持现状
      • 竞争力低、行业吸引力中：分阶段撤退
      • 竞争力高、行业吸引力低：收回投资
      • 竞争力低、行业吸引力低：放弃
      • 竞争力中、行业吸引力低：分阶段撤退
    • STP 营销战略
      • 市场细分
      • 选定目标细分市场
