    排序

    • cpu 排序： 先输入 o，然后输入 cpu
    • 内存排序： 先输入 o，然后输入 rsize
    • P：按照 cpu 使用率排序
    • M：按照内存使用率排序
    • l：切换显示平均负载和启动时间信息
    • m：切换显示内存信息
    • t：切换显示进程和 cpu 状态信息
    • c：切换显示命令名称和完成命令行信息
